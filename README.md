# Dynamic Window Manager

### Dependencies

```sh
sudo pacman -S base-devel xorg-server xorg-xinit xorg-xinput libinput libx11 libxinerama libxft alacritty feh dex ttf-firacode-nerd
```

### Installation

```sh
cd dwm
sudo make clean install
```

### Recommended

- [dmenu](https://github.com/le0flo/dmenu)
- [slstatus](https://github.com/le0flo/slstatus)
